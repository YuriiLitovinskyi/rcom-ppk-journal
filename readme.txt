Утиліта для виводу журналу по номеру ппк в РКОМі.
Запускається на хості з РКОМом.
РКОМ та Монго повинні бути запущені.

Програма зчитує всі останні дані з бази РКОМу по номеру введеного ППК, і виводить у форматах .txt, .xlsx, .html.
Увага! Журнал, отриманий за допомогою даної програми, є лише допоміжним! В першу чергу орієнтуватися потрібно на журнал з ПЦС.
